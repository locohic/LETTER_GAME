#DESCRIPTION DE NOTRE ARCHITECTURE
Notre projet est subdivisé en trois parties correspondant à des répertoires. Il s'agit de :
.src/main/java: comprend les différents packages et les différentes classes associées.
.src/main/ressources: comprend les différents fichiers du projet
.src/test/java: comprend toutes les classes des différents tests effectués.

#RESPONSABILITE DES PACKAGES
*Le package *fr.esiea.unique.ahingora.nadeswaran* est le package qui comprend les classes principales comme le main.
*Le package *fr.esiea.unique.ahingora.nadeswaran.letterpot* est le package qui comprend la classe du pot commun et les méthodes associées.
*Le package *fr.esiea.unique.ahingora.nadeswaran.letterbag* est le package qui comprend la classe du sac et les méthodes associées.
*Le package *fr.esiea.unique.ahingora.nadeswaran.dictionary* est le package qui comprend la classe du dictionnaire et les méthodes associées.
